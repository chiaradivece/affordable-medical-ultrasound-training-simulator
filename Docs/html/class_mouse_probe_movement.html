Detailed Description

Move the probe using the mouse.

Controls are:

  • RIGHT CLICK (or alt-click) to position the probe
  • CLICK AND DRAG to rotate the probe in place

Member Data Documentation

bool MouseProbeMovement.dragMode
private

"Drag mode" is turned on when the user is click-dragging to rotate the probe.

When not in drag mode, the user is free to use the mouse to interact with other parts of the scene (e.g. GUI elements).
