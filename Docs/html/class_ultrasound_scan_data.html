Detailed Description

All the information collected by an UltrasoundProbe in a single frame.

It can be iterated over, yielding the individual UltrasoundScanline objects that make up the data.

The list of scanlines.

This is kept semi-immutable to outside classes - members can be added, but not removed. The only accessor for scanlines returns a readonly view of it. Individual scanlines can be modified using this class's iterator.
