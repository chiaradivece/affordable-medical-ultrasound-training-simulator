Detailed Description

The MonoBehaviour script to attach to the GameObject representing an Ultrasound Probe.

Used to set various configuration properties through the inspector.

This script does not actually perform any of the ultrasound rendering logic. The primary entry point into the HORAY scan is in the HorayProbeOutput script.
