float [] GrayscaleGaussianBlur.ApproximateGaussianCoefficients ( int  count)

Uses an approximation method for calculating a normal distribution based on this paper (tweaked a bit so that it is centered at 0 and gives values < 1)

“A cosine approximation to the normal distribution” D. H. Raab, E. H. Green, Psychometrika, Volume 26, pages 447-450.

Parameters
countHow many Gaussian terms to calculate (in addition to the 0th one, which is always 1.0).
